# Parachute
パラシュートの部品と設計図をまとめたディレクトリです.

## パラシュートの部品/材質
|使用する部品|使用する材質|サイズ|
|----|-----|
|パラフォイル|ナイロンタフタ/リップストップナイロン|1500mm四方|
|パラコード|ナイロン紐|各700mm|
|ハトメ|真鍮製|直径8mm|

## 製作行程
1. 青細線の通りに布を切り出す.
2. 黒線に一致するように正六角形の端の長方形を二回折り返して縫う.
3. 青太線の通りに中心を切り落とす.
4. 正六角形の各頂点にハトメにあったサイズの穴(8mm)を開け、ハトメで補強する.
